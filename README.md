# Jellyfin Plugin Repository

Auto-synced Jellyfin plugin manifest via GitHub Actions.

## Plugins

| Plugin | Upstream | Description |
|--------|----------|-------------|
| StrmTool | [jinlin-teck/StrmTool](https://github.com/jinlin-teck/StrmTool/tree/jellyfin) | Extract media info from strm files to speed up playback |

## Usage

1. Go to **Dashboard → Plugins → Repositories → Add**
2. Paste the manifest URL shown below
3. Save, then check **Catalog** to install/update

## Auto Update

GitHub Actions checks for upstream releases daily. Manual trigger is available in the Actions tab.
